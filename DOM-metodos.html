Um método é uma função associada a um objeto, ou, simplesmente, um método é uma propriedade de um objeto que é uma função. Métodos são definidos de uma forma que as funções normais são definidas, exceto que eles tenham que ser aribuidos como propriedades de um objeto. FONTE: MDN Web Docs
